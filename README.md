# Candidate ranking AI -

This project will be an AI-powered recruitment backend API built using Golang.

This is my first project combining AI, databasing and REST Golang APIs together. 
The end product should allow recruiters to:
- Create a job posting,
- Upload multiple candidate CVs
- Analyse the uploaded CVs using AI
- Rank the candidates based on the job requirements
- Generate hiring insights

# Tech Stack:
- Golang
- PostgreSQL
- Docker
- OpenAI API
- JWT Authentication

I also plan to use AI tools such as Claude and ChatGPT to guide me through any potential setbacks/issues which may arise.

The project is currently in development however the features I plan to implement are:
- User authentication
    Only allowing recruiters to create and see their own jobs, candidate analyses, etc
- Job posting management
- PDF CV upload
- Resume text extraction
- AI-Powered candidate analysis
- Candidate ranking
- Docker support
- Unit tests
